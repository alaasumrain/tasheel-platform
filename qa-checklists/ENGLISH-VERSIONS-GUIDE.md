## Quick Checklist for English Pages

For each English page, verify:

### Translation
- [ ] All text in English
- [ ] No Arabic text visible
- [ ] Professional English writing

### Layout
- [ ] LTR layout (`dir="ltr"`)
- [ ] Content aligns left
- [ ] Icons positioned for LTR
- [ ] Spacing adjusted for LTR

### Functionality
- [ ] Same functionality as Arabic version
- [ ] Links work correctly
- [ ] Forms work correctly

### Responsive
- [ ] Mobile: Single column
- [ ] Tablet: Appropriate layout
- [ ] Desktop: Optimal layout

## Template Usage

1. Copy `TEMPLATE.md`
2. Rename to match page (e.g., `en-about.md`)
3. Update:
   - Page route
   - Component path
   - Page purpose
   - Translation section (English instead of Arabic)
   - Layout section (LTR instead of RTL)
4. Add page-specific features
5. Update `00-INDEX.md`
